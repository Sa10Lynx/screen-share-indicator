## Questions & Answers

**Q: Why do we use React instead of vanilla JavaScript?**  
A: React makes it easier to manage state (data that changes). When state changes, React automatically updates the UI.

**Q: What is Tailwind CSS?**  
A: Tailwind is a CSS framework that gives you ready-made utility classes. Instead of writing custom CSS, you use class names like `bg-blue-500` to style elements.

**Q: How do animations work?**  
A: We defined animations in `tailwind.config.js` (pulse-slow, pulse-fast). Then we apply them with class names. Tailwind handles the CSS.
